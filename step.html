Github Page Steps

Bagaimana sih cara buatnya ??

LANGKAH PEMBUATAN GITHUB PAGES

> 1

Pastikan memiliki akun github

Jika sudah memiliki akun, Login ke akun Github.

> 2

Membuat repositori baru

Selanjutnya adalah membuat repository dengan klik Repository → New → Masukkan nama repository untuk githubpages (username.github.io) → Klik Create repository

> 3

Melakukan git pada lokal directory

Pastikan sudah menginstal Git kemudian mencari folder yang berisi file-file website yang akan dibuat, lalu klik kanan pada folder → Pilih Git Bash Here

> 4

Inisiasi repository lokal

Lakukan inisisasi repositori dengan menuliskan perintah 'git init' pada gitbash

> 5

Masukkan semua file kepada stagging area

Untuk memasukkan file-file kepada stagging area, tuliskan perintah 'git add .' titik (.) di sini artinya mengacu ke semua file.

> 6

Lakukan commit

Untuk melakukan commit, gunakan perintah 'git commit -m"first commit"'. Pada apitan tanda petik ialah merupakan pesan dari commit, sehingga dapat diubah menjadi kalimat apapun sesuai apa yang dicommit.

> 7

Melakukan remote repositori github

Untuk melakukan remote repositori github, tuliskan perintah 'git remote add origin https://github.com/username/username.github.io'. url https dapat diubah sesuai dengan nama repositori yang ingin di remote

> 8

Push file direktori lokal ke remote repositori

Untuk memasukkan file lokal ke remote repositori, gunakan perintah 'git push origin master'. Lalu akan muncul tampilan untuk memasukkan token authorization.

> 9

Salin token dan push berhasil

Untuk mendapatkan token authorization dapat kembali ke halaman GitHub, lalu klik setting dan pilih Developer settings → Personal access tokens → klik Generate token → Masukkan password Github → Salin token → klik Sign In → File sudah berhasil di-push → maka GitHub Pages berhasil dibuat
